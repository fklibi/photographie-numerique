Les métadonnées EXIF

Chaque photo numérique contient des métadonnées cachées (EXIF)

Qu'est-ce que l'EXIF ?

EXIF signifie Exchangeable Image File Format. Ce sont des données enregistrées automatiquement par l'appareil photo au moment de la prise de vue. Elles sont intégrées directement dans le fichier image.

Informations contenues dans les données EXIF

  • Date et heure de la prise de vue
  • Marque et modèle de l'appareil photo
  • Réglages : ouverture, vitesse d'obturation, sensibilité ISO
  • Distance focale de l'objectif
  • Utilisation ou non du flash
  • Coordonnées GPS (géolocalisation)
  • Orientation de l'image
  • Résolution de l'image

Pourquoi les métadonnées EXIF sont importantes

  1. Pour les photographes : comprendre quels réglages donnent les meilleurs résultats.
  2. Pour le classement : trier les photos par date, lieu ou appareil.
  3. Pour la preuve : vérifier l'authenticité et l'origine d'une photo.
  4. Pour la vie privée : attention, les données GPS peuvent révéler votre localisation !

Exemples de données EXIF

Champ EXIF Exemple de valeur Signification
Make Canon Marque de l'appareil
Model EOS 80D Modèle de l'appareil
DateTime 2025:03:15 14:30:00 Date et heure de la photo
ExposureTime 1/250 Vitesse d'obturation
FNumber f/5.6 Ouverture du diaphragme
ISOSpeedRatings 400 Sensibilité du capteur
FocalLength 50 mm Distance focale
GPSLatitude 48.8566° N Latitude GPS

Vie privée et métadonnées

Les métadonnées EXIF peuvent poser des problèmes de vie privée. Quand vous partagez une photo en ligne, les données GPS intégrées peuvent révéler l'endroit exact où la photo a été prise, y compris votre domicile.

Risque Explication Solution
Géolocalisation Les coordonnées GPS révèlent votre position Désactiver le GPS de l'appareil photo
Identité Le nom de l'appareil peut être lié à vous Supprimer les EXIF avant de partager
Habitudes Les dates et lieux montrent vos déplacements Utiliser un outil de suppression d'EXIF

Comment voir les données EXIF

  • Windows : clic droit sur l'image > Propriétés > Détails
  • En ligne : utiliser un site comme exif.tools
  • Python : avec la bibliothèque Pillow (PIL)
  • GIMP : menu Image > Propriétés de l'image
