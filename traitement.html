Le traitement d'image

Le traitement d'image permet de modifier et améliorer les photos

Qu'est-ce que le traitement d'image ?

Le traitement d'image consiste à modifier une image numérique à l'aide d'algorithmes. On peut corriger les couleurs, ajuster la luminosité, appliquer des filtres ou encore détecter des contours.

Opérations courantes

  • Modification de la luminosité et du contraste
  • Correction des couleurs (balance des blancs)
  • Application de filtres (flou, netteté, sépia)
  • Recadrage et redimensionnement
  • Détection de contours
  • Suppression du bruit numérique

Les formats d'image

  1. JPEG : format compressé avec perte, le plus courant pour les photos.
  2. PNG : format compressé sans perte, supporte la transparence.
  3. RAW : données brutes du capteur, permet un traitement maximal.
  4. GIF : supporte les animations, limité à 256 couleurs.
  5. BMP : format non compressé, fichiers volumineux.

Comparaison des formats

Format Compression Transparence Poids Usage
JPEG Avec perte Non Léger Photos, web
PNG Sans perte Oui Moyen Logos, captures
RAW Aucune Non Très lourd Photo pro
GIF Sans perte Oui (1 bit) Variable Animations

Le codage des couleurs

En numérique, chaque pixel est codé en RVB (Rouge, Vert, Bleu). Par exemple :

Couleur Rouge (R) Vert (V) Bleu (B) Code hexadécimal
Blanc 255 255 255 #FFFFFF
Noir 0 0 0 #000000
Rouge pur 255 0 0 #FF0000
Vert pur 0 255 0 #00FF00

Logiciels de traitement d'image

  • GIMP : logiciel libre et gratuit, alternative à Photoshop
  • Adobe Photoshop : logiciel professionnel payant
  • Darktable : logiciel libre pour le traitement de fichiers RAW
  • Paint.NET : logiciel gratuit pour Windows
